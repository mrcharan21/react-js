üìå What is JavaScript?

  • JavaScript is a client-side scripting language.
  • Used as a programming language to make websites dynamic.
  • Maintained by the ECMAScript (ECMA) organization.
  • Popular libraries and frameworks:
    • React.js
    • Angular.js
    • Vue.js
    • Next.js
  • Fast-loading in browsers.
  • Supports dynamic content and user interaction.
  • Run using:
    • <script> tags (internal/external)
    • In <head> or <body> sections
    • Browser console or Node.js terminal
  • Supports REPL (Read, Evaluate, Print, Loop)
  • File extension: .js
  • JavaScript is case-sensitive
  • Default async loading behavior

üì¶ Variables in JavaScript

  • Variables are containers to store data.
  • Examples: var a = 10;, let b = 10.5;, const c = "Charan";
  • Rules:
    • No spaces in variable names
    • Can't use reserved keywords
    • Don't use reserved function names
  • var: Redeclared and reassigned
  • let: Reassigned but not redeclared
  • const: Neither reassigned nor redeclared
