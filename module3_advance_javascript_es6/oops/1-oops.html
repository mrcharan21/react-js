What is opps ?

oops stands for object oriented programming structure langugage


oops provide some features


features of oops

Inheritance : one class can inherit properties and methods of another class


Polymorphism : one interface can be implemented by multiple classes


Abstraction : hiding the implementation details and showing only the public interface of a class


Data hiding : hiding a data inside a class and providing only public methods to access and modify the data


Encapsulation : hiding the internal details and provide only neccessary methods and properties to the outside world
